# Product Listing App 🚀

A React application for viewing products with **search**, **category filtering**, **sorting** (price/rating), and **add to cart** functionality.



## Features ✨

- ✅ **Product Grid**: Responsive display of products (image, name, price, category, rating)
- ✅ **Search**: Filter products by name (real-time)
- ✅ **Category Filter**: Dropdown with "All" + dynamic categories
- ✅ **Sorting**: Price (low-high/high-low), Rating (high-low)
- ✅ **Add to Cart**: Button logs product name to console
- ✅ **Responsive Design**: Works on mobile/desktop


## Tech Stack 🛠️

- React 18 (functional components + hooks)
- CSS (custom responsive styles)
- Mock JSON data
